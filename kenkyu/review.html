高分子樹脂内での物理的微小発泡現象のモデリング

目的

ポリマー発泡は産業的に非常に重要な材料である。 低い密度にあっては、ガラスファイバーのようなよく知られた絶縁体と比較して、 よい熱的絶縁特性を持っている。 また、硬さが要求されるような高い密度にあっても、場合の材料節約となる。

マイクロセルラーは、10μmオーダーのポリマー発泡体と定義されている。 その発泡工程は、主に二つのプロセスからなる。

  • 加圧容器(RAMESHらは35MPa)の中に原料ポリマーを置き、 窒素などの非反応性ガスで飽和させる。 この段階で、温度はガラス転位点(ポリスチレン:373K)以下となっている。
  • 加圧容器から、常圧の高温バスの中へ移動する(ここでは減圧と加熱をほぼ同時に行っているが、 KUMARらは、 加熱のタイミングを適当に遅らせることによって、素材表面での発泡を押えられると報告している)。 ガスは過飽和状態となり、非常に不安定になる。 液体となって、やわらかくなったポリマー中に、 無数の気泡の「核」が生れ、成長が始る。
この発泡のプロセスは、さらに核生成過程と発泡過程にわけられる。 本レビューでは、古典的な相転位モデルを拡張したColtonとSuhのモデルと、 Rameshらによる、 材料中に始めから存在しているmicrovoidを考慮した核生成モデルを 比較・検討してみたい。

相転位でモデル化した気泡核生成過程

Suhらは、気泡の核生成には、ポリマーの状態によって、3種類のメカニズムを 提示している。それらは、ポリマーの主相に解けにくい添加物の量によって、 均質系核生成,不均質系核生成,混合系核生成と場合分けされる。
　均質系核生成(Homogeneous Nucleation)は、添加物の量が、 主成分、添加物が共になって一つの均一な相を形成しているような場合に 起きるとされる。
　不均質系核生成(Heterogeneous Nucleation)は、二つの成分間の界面上に 新たな相として形成する場合に生じる。
　混合系発泡(Mixed mode Nucleation)は、第二成分の量が、均質/不均質を分ける 臨界量前後で、両発泡が競合して起きるような場合に起きるとしたモデルである。

均質相発泡

気泡を球と扱い、ポリマーと気泡間の界面エネルギーγbpの 等方性を仮定すると、発泡においての、自由エネルギー変化は次のように表せる。
ΔGhom = -VbΔP + Abpγbp
= -(4πr3/3)ΔP + (4πr2)γbp

ここで、ΔPは気泡内ガスの圧力を示しているが、 ガスが理想気体であると仮定すると、飽和圧力で代用できる。
　これを微分すると、自由エネルギー変化が極大値をとる 臨界半径r*が次のように得られる。
r* = 2γbp / ΔP

より、この時の自由エネルギー(臨界自由エネルギー)は
ΔGGhom = 16γ3 / (3ΔP2)

と表せる。ここでガスの濃度 :C0,頻度因子 :f0とおくと、 均質核生成速度N0は
Nhom = C0f0exp( -ΔG*hom/kt )

となる。

不均質相発泡

今、表面エネルギーのバランスをとると、
γap = γbp + γabcosθ
が成り立つ。aは固体微粒子pはポリマー,bは気泡で、 γは、添え字が表す二相間の表面エネルギーを表す。 また、θは濡れ角度を表す。

これより、自由エネルギー変化は

ΔGhet = -Vb + Abpγbp - Aapγap
= ( -(3π/4)r3ΔP + 4πr2γbp )S(θ)
ただし,θのみの関数Sは、
S(θ) = (1/4)(2+cosθ)(1-cosθ)2
であり、常に1以下の値しか取らない。 rで微分すると、先と同様に臨界半径と 臨界ギップス自由エネルギーが次のように得られる。
ΔG*het = ΔG*homS(θ)
が成り立つ。平均的な濡れ角度はθ=20°で、S(θ)は 10-3オーダーとなる。つまり、非均質系核生成のエネルギー障壁は 界面の存在によってはるかに縮められる。

なお、非均質核生成速度 : Nhetは、Nhom同様、

Nhet = f1C1exp( -ΔG*het / kt )
と表せる。

混合系核生成

均質系核生成と非均質系核生成は互いに排他的な関係ではなく、双方が 同時に発生する場合も当然存在する。この場合、均質系核生成は、 先行して起る非均質系核生成によってガスが奪われると考えられるので、 次のように生成速度式は修正される。
N'hom = f0C'0(-ΔG*hom/kT)
ただし、
C'0 = C0 - Nhettnb
であり、tは最初の不均質系核生成が始まってからの時間、 nは気泡核中の分子数を示す。

Microvoidの存在を考慮した非均質核生成モデル


Kweeder,Rameshらは相転位を基礎とした 古典的な核生成モデルが熱プラスチックの 微小規模の発泡(Microcellular Foams)を---特に非均質核生成において--- 十分に説明できない場合があるとして、 ポリマー材料の熱加工時に発生する微小空洞(Microvoid)を 考慮してモデルを考えた。
　彼らは例としてポリスチレン(PS)をベースとしたポリマーに 添加物としてポリスチレンとポリブタジエン(PB)からなるゴム微粒子 (Rubber Particle)を加えた場合の非均質系をとりあげた。
　ポリブタジエンとポリスチレンのガラス転位点はそれぞれ181K,373Kであるので、 ポリマー熱処理後の冷却において、ポリスチレン相が先に固化する。 一方、ポリブタジエン相はそのまま室温に至るまで熱収縮をつづけるため、 ここに、熱膨張係数の違いから、微小な亀裂(cavitation)が発生する。 このうち、弾性抗力に打ち勝つだけの十分な半径を持ったmicrovoidだけが、 発泡の核として残ることができる。

物質収支

ポリマー中の飽和ガスの物質収支を考えると、
発泡後のガス量 = 発泡前のガス量 −(発泡後の気泡ガス−microvoid中のガス)
が成り立つ。ポリマー中に含まれるガス量をヘンリー定数を用いると、 次のように式を変形できる。
k1P = k0Ps - 4πTr/3VpPr [ R3P/Tf - R03Ps/Ts ]
Tr,Prは参照温度と圧力,Ts,Psは飽和温度と圧力を表す。 Vpは気泡を取り囲むポリマーの圧力をしめしている。 ヘンリー定数k0,k1は
k0= m'Ps + c0
k1= m'P(R)+ c1

と表せるので、式(\ref{eq:ramesh-7})は、圧力Pの二次方程式になる。 これを解くと、
P(R)=-B+[B2+4m'(k0+k3)Ps] 1/2 / 2m'
B =c1+k2R3
k2=4πTr/3VpPrTf
k3=4πR03Tr/3VpPrTs

表面張力と弾性効力

弾性応力を考えるにあたり、無限のポリマーの海の中に一つの気泡を考える モデルと、一つの気泡を有限量で球殻(Envelop,Shell)状で 取り囲むポリマーだけを考えるセルモデルの二種類がある。 前者は、後者のポリマー殻の量を無限大とおけばよい。
応力の半径方向のつりあいを考えると
ΔP = Pg-Pf
=2σ(T)/R - 2∫SR(τrr-τθθ)(dr/r)

ただし、境界条件
r=R :σrr= -Pg+2τ/R
r=S :σrr= Pf

ここで、Rは気泡半径を表す。Sは単位セルの半径である。 σ(T)は表面張力の大きさで、質量密度ρ(T)を使って、
σ(T) = σ(298)(ρ(T)/ρ(298))4
と表せる。フックモデルを用いると、ΔPは次のように導き出される。

ΔP= 2σ(T)/R + 2Gc[ (1/4){(y+R3)/(y+R30)}-4/3
+ {(y+R3)/(y+R30)}-1/3
- (1/4){R3/R03}-4/3 - (R3/R03)-1/3]
ただし、yはy=S3-R3を表し、4πy/3が一気泡あたりのポリマー量を示す。 式(\ref{eq:ramesh-7})から計算された圧力が、表面弾性力を超過すれば、 空洞は気泡の核となりえる。その境界となる(臨界)半径は計算機などで 求めることができる。
その半径分布は対数正規分布となっているので、 半径Rより大きい空洞の数は、単位体積あたりの空洞の数:N0を使って
N= N0[ 1 - 1/(σ(sπ)1/2) ∫R-∞ {-(log R/ξ)2/2σ2 }d log R ]
= N0/2 erfc(log R/ξ)/σ√2)

とあらわせる。
実験によると、発泡数とゴム微粒子の数は非常に相関関係があるので、 総数をゴム微粒子の数から見積ることができる。 これはゴム微粒子が多数のvoidを持っていても、もっとも大きいものだけが残り、 他のより小さいmicrovoidはガスを奪われて消えてしまうからである。

考察

古典的な相転位理論を拡張したColton,Suhらのモデルでは、ステアリン酸亜鉛 添加物の量に対する核生成密度の変化を定性的にはうまく表しているものの、 実測値と理論値との間のずれが大きい。
また、非均質系核生成に関して、 古典的な核生成理論では臨界核のサイズが小さく、 5--20 Aオーダーになってしまう場合もある。
一方、Rameshらのモデルは、飽和圧力,発泡温度の変化に対して、 定量的にもよく合致しているが、特に一気泡あたりのポリマー量を有限と 置いたほうがよくあっている。
